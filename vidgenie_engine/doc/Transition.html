    class Transition

    Represents a transition between 2 stacks. Currently the transition object is only aware of the previous stack so we aren't able to apply a transitioning effect to the underlying stack.

    Transition Effects¶ ↑

    The following transition effects are supported

    - Fade
- SlideLeft
- SlideRight
- SlideUp
- SlideDown
